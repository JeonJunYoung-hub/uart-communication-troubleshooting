# UART-Based LED Matrix Visualization System

Real-time environmental data visualization system between
Arduino (Sensor MCU) and Metro M4 (Display Controller).

## System Focus

- Deterministic packet-based UART communication
- Asynchronous stream stabilization
- Inter-device timing synchronization
- Memory-aware LED display optimization
